The contact sidebar shows a complete profile for whoever you're emailing -- name, title, company, LinkedIn, upcoming meetings, email history, investor data, and your stored memories. All assembled automatically.

## Opening the Sidebar

Press `p i` (panel: info) while viewing a thread. The sidebar stays open as you navigate, updating for each thread's contact.

## What You See

<AccordionGroup>
  <Accordion title="Identity & Company">

    - **Name, title, one-line bio**
    - **Company name, logo, industry, size, description**

    - **Location** with map pin

  <Accordion title="Investor Information">

  <Accordion title="Activity">
    - **Upcoming meetings** -- calendar events with this contact
    - **Recent threads** -- your latest email conversations (click to jump)

  </Accordion>
</AccordionGroup>

## 9-Provider Enrichment Waterfall

Slashy queries up to nine data providers in a prioritized waterfall. When a new contact enters your inbox, enrichment triggers automatically in the background.

| Status | Meaning |
|--------|---------|
| **Enriched** | Profile populated with data |
| **Pending** | Enrichment in progress |
| **Failed** | No data found across all providers |
| **Not attempted** | Not yet enriched |

Results from multiple providers are merged to build the most complete profile possible.

## Bookface Investor Matching

For YC ecosystem users, Slashy matches contacts against Bookface investor data using four strategies (in confidence order):

<Steps>
  <Step title="Email domain match">Contact's domain matches a known fund domain</Step>
  <Step title="Exact name match">Name exactly matches a known investor at the same domain</Step>
  <Step title="Fuzzy name (same domain)">Fuzzy name match within the same domain</Step>
  <Step title="Fuzzy name (global)">Fuzzy name match across all known investors</Step>
</Steps>

<Info>Investor profiles appear automatically for whitelisted users when viewing email from a matched investor.</Info>

## Contact Autocomplete

Enrichment data powers compose autocomplete. Suggestions in To/CC/BCC fields are ranked by **interaction frequency**, **recency**, and show the contact's name, email, title, and company.

| Action | Shortcut |
|--------|----------|
| Open contact sidebar | `p i` |
| Toggle right sidebar | `Cmd+.` |
| Open AI chat panel | `p a` |
