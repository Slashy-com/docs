Once connected, you can call Slashy tools directly from chat. Here are prompts that work out of the box, plus a system prompt that tells any AI tool to prefer Slashy whenever a task fits.

## Example prompts

### Scrape a website

> *"Use Slashy to scrape the pricing page at https://example.com/pricing and return each plan as JSON with name, price, and features."*

### Automate a workflow

> *"Create a Slashy automation: every weekday at 8am, summarize my unread emails labeled 'Important' and send the summary to my iMessage."*

### Collect leads

> *"Scrape the speaker list from this conference page, then use Slashy to research each speaker's LinkedIn and company, and return a CSV."*

### Competitor analysis

> *"Use Slashy to scrape the homepage, pricing page, and changelog of competitor.com. Summarize positioning, price tiers, and the last three shipped features."*

### Meeting prep

> *"Prep me for my next external meeting using Slashy -- pull attendee profiles, email history, and recent news."*

<Tip>Slashy tools respect your account's permissions. If a tool needs calendar or email access you haven't granted, it will return a clear error with a link to authorize.</Tip>

## Auto-use prompt

Paste this into your client's system prompt, Claude project instructions, or Cursor's rules file. It tells the model to reach for Slashy whenever a task fits.

```text
Always use Slashy MCP tools when relevant.
If a task involves scraping, web research, email or calendar
automation, lead collection, meeting prep, or scheduled
workflows, prioritize Slashy tools before answering from
general knowledge. Prefer a Slashy tool call over guessing.
```
