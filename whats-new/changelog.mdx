Stay up to date with the latest Slashy features and improvements. Updates ship continuously -- this page covers the highlights.

---

## April 2 -- 6

- **HubSpot integration** -- Connect Slashy to HubSpot to sync contacts and log email activity to CRM records automatically.
- **Outlook-style split view** -- New split layout with reorganized list items, inbox indicator, and smoother sidebar resizing.
- **Select text and Enter to quote** -- Highlight any passage in a thread and press Enter to start a reply that quotes just that selection.
- **Inline reminders UI** -- When you ask the AI Agent to list reminders, results now render as an interactive inline list instead of plain text.
- **Mark done from thread detail** -- Toggle mark done / mark not done directly inside the thread detail view.
- **Shareable automation templates** -- Share any automation with a URL so teammates can import it in one click.
- **BCC in snippets** -- Snippets can now include BCC recipients alongside To and Cc.
- **Desktop app step in onboarding** -- New onboarding flow prompts you to download the desktop app with Mac universal links.
- **Wrong-account sign-in hint** -- Auth flow now detects when you're signing in with your inbox email and points you to the right account.
- **"Sent with Slashy" on mobile** -- Mobile-sent emails include the Slashy branding footer, matching desktop.
- **Avatars in mobile detail view** -- Contact avatars now appear in the mobile thread detail with a waterfall layout, shown by default for every conversation.
- **Mobile compose & reply settings** -- Compose and reply preferences are now available in the mobile settings sheet.
- **Improved search ranking** -- Search results now factor in thread length, with recent threads prioritized when relevance is close.
- **Faster mobile cold start** -- Delta sync and body warming are deferred until the thread list loads, so the inbox appears sooner.

---

## March 30 -- April 1

- **Inline reply from macOS notifications** -- Reply-all directly from notification banners without opening the app.
- **Mobile quoted content toggle** -- Show or hide the original message in reply compose with a three-dot toggle, matching the desktop experience.
- **iOS push notification layout** -- Cleaner three-line notifications showing sender name, subject, and snippet.

---

## March 23 -- 29

- **Propose new time for calendar invites** -- Organizers can propose alternative times for events. Attendees see proposal indicators and can accept, decline, or counter-propose.
- **Multi-attendee availability checking** -- When scheduling, Slashy can now check multiple attendees' calendars at once and return mutual free slots.
- **Automations filter** -- Filter dropdown in the automations tab to separate recurring automations from one-off reminders.

---

## March 14 -- 22

- **Email rating: missing recipient detection** -- The email rating system now flags when key recipients are missing from To or Cc fields before you send.
- **Sender timezone hints** -- AI drafts now factor in the sender's timezone for more context-aware responses.
- **Google-first profile pictures** -- Contact profiles now prefer Google profile photos for more accurate avatars.
- **Stable agent file links** -- Files shared by the AI Agent use stable URLs that don't expire, preventing broken links in conversations.
- **Name-only contact research** -- The AI Agent can now research contacts using just a name (no email required), with smart disambiguation for multiple matches.

---

## March 1 -- 13

- **Claude Sonnet 4.6 upgrade** -- Slashy's AI model upgraded to Claude Sonnet 4.6 for better drafting and agent responses.
- **Self-contained thread list** -- Major performance improvement: the inbox now loads full thread metadata in a single call, making list views significantly faster.
- **Smarter iMessage replies** -- iMessage replies now include full thread context for more accurate agent responses.
- **Better email editing context** -- Email edits now use thread context and shared memory for more personalized suggestions.

---

## February 16 -- 28

- **Mobile push notifications** -- Full push notification pipeline for the Slashy mobile app via Expo, triggered for important emails.
- **AI auto-reminders** -- When enabled, Slashy analyzes each sent email and automatically schedules follow-up reminders with AI-determined timing. Covers business requests, cold outreach, scheduling, and more.
- **Booking page cancel and reschedule** -- Guests can now cancel or reschedule bookings without a Slashy account using public endpoints.
- **Teams, thread sharing, and comments** -- Share email threads with teammates, leave threaded comments with @mentions, and get real-time notifications. Shared threads auto-import from Gmail.

---

## February 1 -- 15

- **Default conferencing preference** -- Set a default video conferencing type (Google Meet, Zoom, or custom link) for new calendar events. The AI Agent respects this preference automatically.
- **Optional calendar attendees** -- The AI Agent can now mark attendees as optional when creating calendar events.
- **Per-label All Mail control** -- Choose which importance labels show threads in the All Mail view.
- **AI draft tracking** -- Slashy now tracks when you've used an AI-generated draft, so the auto-fill UI doesn't re-appear.

## Submit Feature Requests

- **Direct message** the Slashy team via iMessage, WhatsApp, Telegram, or Slack
- **In-app support** -- describe the feature and why your current workflow falls short
- **Tell the AI Agent** (`Cmd+Shift+L`) -- the team reviews agent conversations for common requests

<Tip>The best feature requests include: what you are trying to do, how you currently work around it, and how often you hit the problem.</Tip>
